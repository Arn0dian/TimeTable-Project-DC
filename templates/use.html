How To Use Our Time Table Generator

Follow the following steps for generating your Time Table

  1. You have to enter Two CSV :-
      2.   (a) Faculty CSV file - This file will contain information about all the faculties in an institution.
        (b) Courses CSV file - This file will contain information about all the courses taught by the respective faculties.

  2. Press the Generate Button to start your generation.

  3. Hooray, you will get your generated time table.

  4. You can now download it in csv format.
